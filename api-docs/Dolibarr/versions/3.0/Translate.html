Load($domain, $alt = 0, $stopafterdirection = 0, $forcelangdir = '')

Load translation key-value for a particular file, into a memory array.

If data for file already loaded, do nothing. All data in translation array are stored in UTF-8 format. tab_loaded is completed with $domain key. Value for hash are: 1:Loaded from disk, 2:Not found, 3:Loaded from cache
